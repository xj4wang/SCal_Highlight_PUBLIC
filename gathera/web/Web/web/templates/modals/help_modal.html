{%block help-modal-title%}How to use Gathera{%endblock%}
{%block help-modal-body%}

Gathera is a system for efficient high-recall retrieval that allows retrieving and assessing relevant documents and provides high data processing performance with a user-friendly document assessment interface.

There are two retrieval components in Gathera.

Search: the search component allows you to search for documents using a search engine. Judgments made in the search component are used to train the Discovery model.

Discovery: Discovery is an iterative feedback process that uses a machine learning classifier to train on relevance judgments made by the user. The Discovery interface presents the user with top scoring unjudged documents in the collection. After each iteration of judging and re-training, the learning model improves and returns the next most likely relevant documents to judge.

To learn more about Gathera, visit the Gathera's website.
